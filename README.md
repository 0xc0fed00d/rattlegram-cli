
### Rattlegram

Transceive UTF-8 text messages with up to 170 bytes over audio in about a second!

Encode usage:
```
./encode MESSAGE CALLSIGN [NOISE_SYMBOLS] [CARRIER_FREQUENCY] [RATE] [BITS] [CHANNEL]
```

Decode usage:
```
./decode FILE CHANNEL
```
#### Dependcies
AudioFile https://github.com/adamstark/AudioFile
